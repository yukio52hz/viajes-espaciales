Un grupo de científicos

Le ha solicitado que realice una aplicación que permita tener la información de los planetas o cuerpos celestes de los que se tiene conocimiento.

Dentro de las funcionalidades debe tener la de registrar un cuerpo celeste, para ello primero se debe seleccionar si el cuerpo celeste es una estrella, planeta o satélite.

Los científicos además buscan mantener una bitácora de las misiones realizadas a diferentes lugares del universo para eso se debe entender que un conjunto de misiones hacen parte de un programa espacial.Por lo tantotamnien debe almacenarce la información de los programas y misiones.

Para elaborar el proyecto se utilizo css,html,bootstrap y js

Nota: El proyecto funciona con localStorage asi que si los datos no cargan,refresque la página

Detalles del proyecto

Un grupo de científicos le ha solicitado que realice una aplicación que permita tener la información de los planetas o cuerpos celestes de los que se tiene conocimiento. Dentro de las funcionalidades debe tener la de registrar un cuerpo celeste, para ello primero se debe seleccionar si el cuerpo celeste es una estrella, planeta o satélite. Tanto una estrella como satélite o planetas son considerados cuerpos celestes.

Tambien que se puenda almacenar la información de los programas espaciales y misiones

Nota: El proyecto funciona con localStorage asi que si algo no se vizualiza bien refresque la página
